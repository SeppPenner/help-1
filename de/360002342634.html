DHCP Server - Es kann nur einen geben!

Wenn Sie den DHCP Server des eBlockers aktivieren, denken Sie bitte daran auch den DHCP Server Ihres Routers zu deaktivieren.

Es kann nur einen DHCP Server im Netzwerk geben.

Wenn Sie den DHCP Server des eBlocker deaktivieren, denken Sie bitte daran, dass Sie vorher den DHCP Server Ihres Routers wieder aktivieren.

Beim Wechsel des DHCP Servers müssen immer alle Geräte im Netzwerk einmal vom Netzwerk getrennt und wieder verbunden werden. Entweder trennen Sie einmal kurz das Netzwerkkabel, deaktivieren ein mal kurz das WLAN, oder starten das Gerät einmal neu.
