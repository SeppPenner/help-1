Kein eBlocker Symbol auf setup.eblocker.com

Wenn Sie beim Aufruf der Seite http://setup.eblocker.com/ kein eBlocker-Symbol oben rechts im Browser sehen, löschen Sie Ihren Browser Cache und rufen Sie die Seite http://setup.eblocker.com/ ein erneutes Mal auf.

Wie Sie Ihre Browser Cache leeren, haben wir für Sie in unseren FAQs im Bereich „eBlocker – Allgemeine Fragen“ beschrieben.

Sehen Sie das eBlocker Symbol weiterhin nicht, rufen Sie die Seite http://setup.eblocker.com/ mit einem anderen Gerät auf. Wenn Sie dort das eBlocker Symbol oben rechts im Browser sehen, könnte die Anzeige des eBlocker Symbols in den eBlocker „Einstellungen > Menü Geräte“, für das andere Gerät möglicherweise deaktiviert sein.

Bitte beachten Sie auch unseren Hinweis zum Thema WLAN Repeater in den eBlocker FAQs im Bereich „eBlocker – Netzwerk“.

Sehen Sie nach, ob Ihr benutztes Gerät (der Rechner) eventuell per LAN und WLAN in Ihrem Netzwerk ist. Sind beide Module aktiviert, hat Ihr Rechner eventuell auch Zwei MAC- und IP-Adressen. Wenn Sie das WLAN beispielsweise nicht benötigen, weil Sie ohnehin mit dem LAN Kabel an Ihrem Netzwerk angeschlossen sind, deaktivieren Sie einmal das WLAN bei Ihrem Gerät (Rechner).

Überprüfen Sie, ob Ihr Gerät (Rechner) den eBlocker als Gateway benutzt.

macOS

Sie können mit den Bordmitteln von macOS ganz leicht erkennen, ob der eBlocker richtig eingebunden ist.

Öffnen Sie die Suche (die Lupe oben rechts) und schreiben Sie „Netzwerk“. Die Suche wird automatisch den Text „Netzwerkdienstprogramm“ vorschlagen. Drücken Sie die Enter-Taste. In dem nun geöffneten Fenster, wählen Sie „Trace“ aus, geben im Textfeld www.eblocker.com ein und klicken auf den Button „Trace“. Der Befehl „Trace“ zeigt Ihnen den Weg durch das Internet auf, welchen Sie bis zum Zielserver zurücklegen. Wichtig ist die erste IP-Adresse. Diese sollte die IP-Adresse Ihres eBlockers sein.

Windows

Sie können auch mit den Bordmitteln von Windows ganz leicht erkennen, ob der eBlocker richtig eingebunden ist.

Öffnen Sie bitte das Kommandozeilen Programm (CMD). Geben Sie nun den Befehl „tracert www.eblocker.com“ ein und warten Sie kurz, bis die Anfrage beendet ist. Der Befehl „tracert“ zeigt Ihnen den Weg durch das Internet auf, welchen Sie bis zum Zielserver zurücklegen. Wichtig ist die erste IP-Adresse. Diese sollte die IP-Adresse Ihres eBlockers sein.
