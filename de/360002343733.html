Release Notes 1.3

Neue Funktionen

Jugendschutz (Nur Family Edition)

Die Jugendschutz-Funktionen wurden erweitert: Es ist jetzt möglich individuelle Benutzerkonten für alle Familienmitglieder anzulegen. Jedem Benutzer können dann ein Jugendschutz-Profil sowie die jeweils verwendeten Geräte zugeordnet werden. Während jedes Gerät einen Hauptnutzer hat, kann man Benutzern erlauben auch andere Geräte durch Eingabe eines Benutzer-PINs vorübergehend in Besitz zu nehmen.

Zusätzlich zur Einrichtung von Zeiträumen, in denen der Internet-Zugriff gestattet ist, können jetzt für jedes Jugendschutz-Profil maximale Nutzungszeiten pro Tag festgelegt werden.

Darüber hinaus ist es jetzt möglich, selbst eigene Listen von verboten oder ausdrücklich erlaubten Websites anzulegen.

VPN

Der Netzwerk Assistent für VPN-Zugänge wurde verbessert: Es ist jetzt möglich zusätzliche Dateien zu einer Konfiguration hochzuladen, zum Beispiel für Zertifikate oder Schlüssel.

Die Rückmeldung über den Status der VPN-Verbindung während der Startphase wurde verbessert.

Tracker-Blocking / AdBlocking

Für die gerade besuchte Website kann jetzt separat das Ad-Blocking bzw. das Tracker-Blocking ausgeschaltet werden (Whitelisting). Dadurch ist es zum Beispiel möglich, nur das Ad-Blocking abzuschalten – um eine Ad-Blocker-Schranke auf einer Nachrichtenseite zu umgehen – während Sie auf der gleichen Seite weiterhin vor Trackern geschützt sind.

Individuelle Einstellungen für Tracker-Blocking / Ad-Blocking (Nur Family Edition)

In der Family Edition werden die Einstellungen zu freigeschalteten Domains bei Ad-Blocking/Tracker-Blocking (Whitelisting) nun an den individuellen Benutzerkonten gespeichert. So dass jedes Familienmitglied die Einstellungen an die eigenen Bedürfnisse anpassen kann.

Systemmeldungen

Der eBlocker benachrichtigt den Benutzer nun über das Vorliegen von wichtigen Systemmeldungen über das Icon und die ControlBar. Dazu gehört u.a. der zeitweilige Verlust der Stromversorgung oder der Netzwerkverbindung.

Ab diesem Release informiert der eBlocker über das Icon und die ControlBar außerdem über eBlockerOS-Aktualisierungen.
