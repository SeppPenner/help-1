eBlocker DNS Firewall

Wenn Sie diese Funktion aktivieren, können Sie die DNS Anfragen über Ihren eBlocker an eine Liste von verschiedenen DNS Servern verteilen, oder über des Tor-Netzwerk auflösen lassen. Ihnen stehen dafür drei verschiedene Optionen zur Verfügung.

  • Internet Provider
  • Tor-Netzwerk (DNS Auflösung erfolgt anonym)
  • Externe DNS-Server

Internet Provider

Hier wird die Einstellung Ihres lokalen Netzwerkes verwendet. Es wird der DNS Server verwendet, welcher in Ihrem Router eingetragen ist.

Tor-Netzwerk (DNS Auflösung erfolgt anonym)

Hier werden die DNS Anfragen durch das Tor-Netzwerk geleitet und am Ausgangspunkt des Tor-Netzwerkes erst an einem DNS Server übergeben.

Diese Funktion kann den Internetzugriff leicht verlangsamen. Dies gilt auch für Geräte, die sonst nicht Tor nutzen.

Externe DNS-Server

Mit dieser Option können Sie eine Liste von DNS Servern hinterlegen. Diese Liste kann dann nach Verfügbarkeit, der Reihe nach, oder in zufälliger Reihenfolge vom eBlocker abgearbeitet werden.

Eine kleine Anzahl von DNS Servern haben wir hier für Sie aufgelistet.

Weitere öffentlich verfügbare DNS Server finden Sie im Internet. Geben Sie hierfür die Begriffe „Öffentliche Nameserver“, oder „Alternative DNS Server“ in die Suchmaschine ein.
