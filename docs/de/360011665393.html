Information zu der eBlocker HTTPS Funktionen

Folgende eBlocker Funktionen benötigen die Aktivierung der eBlocker HTTPS Funktion, damit der eBlocker diese Funktionen auch bei verschlüsselten Webseiten ausführen kann.

  • Das eBlocker Icon und die eBlocker Controlbar
  • Tarnung
  • Pattern Werbe Blocker
  • Pattern Tracker Blocker
  • Captive Portal Check
  • Kompression
  • WebRTC

Die Aktivierung der eBlocker HTTPS Funktion für ein Gerät mach nur Sinn, wenn Sie auf diesem Gerät auch das eBlocker Zertifikat speichern können.

Geräte bei dem Sie das eBlocker Zertifikat nicht speichern können sind zum Beispiel:

  • Fernseher
  • Internetradios
  • Netzwerklautsprecher
  • AppleTV
  • Amazon Fire TV
  • Internet IP Telefone
