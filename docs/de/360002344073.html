Weder Router noch eBlocker als DHCP Server

Es gibt auch die Möglichkeit, weder den DHCP Server vom Router, noch den DHCP Server vom eBlocker zu nutzen. Sie können auch einen anderen DHCP Server in Ihrem Netzwerk einbinden. So bieten zum Beispiel einige NAS Geräte eine zusätzliche DHCP Server Funktion an. Als weitere Option können Sie sich auch einen eigenen DHCP Server auf Basis eines Einplatinenrechners bauen.
