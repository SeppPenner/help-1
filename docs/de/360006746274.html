Kann man dem eBlocker eine feste IP-Adresse zuweisen?

Ja, diese Funktion bieten die meisten Router in den Netzwerkeinstellungen an.

Bei einem Netgear-Router finden Sie die Einstellungen hier:
Netgear „Erweiterte Ansicht“ > Konfiguration > LAN-Konfiguration > den eBlocker auswählen/bearbeiten.

Ein Fritzbox-Router von AVM bietet die Einstellungen hier an:
Fritzbox „Heimnetz“ > Netzwerk > Reiter „Netzwerkverbindungen“ > den Stift rechts neben dem gewünschten Gerät anklicken > den Haken bei „Diesem Netzwerkgerät immer die gleiche IPv4 Adresse zuweisen“ setzen.

So sollten Sie vorgehen:

  • Öffnen Sie die Einstellungen Ihres eBlockers
  • Öffnen Sie nun die Einstellungen Ihres Routers in einem neuen Browser-Fenster und stellen Sie die IP-Adresse des eBlockers um
  • Gehen Sie zurück zu den eBlocker Einstellungen und rufen Sie das Menü „System“ auf
    Starten Sie Ihren eBlocker neu (im Reiter „Neustart und Ausschalten“)
  • Der Neustart benötigt ca. fünf Minuten
  • Wenn Sie im Browser setup.eblocker.com aufrufen, sehen Sie das eBlocker Dashboard.
  • Die neue eBlocker IP-Adresse sehen Sie auf der Karte "Einstellungen".

Trennen Sie alle aktiven Geräte einmal kurz vom Netzwerk, oder starten Sie sie neu. Das ist erforderlich, damit diese Geräte die neue IP-Adresse vom eBlocker mitgeteilt bekommen.

Wichtig:
Wenn Sie eine andere IP-Adresse für den eBlocker wählen, muss diese im IP-Adressbereich Ihres DHCP-Server liegen! Ändern Sie deshalb nur den letzten der vier Nummernblöcke.

Beispiel: 192.168.1.xxx

Wenn Sie die anderen drei Nummernblöcke ebenfalls ändern, ist der eBlocker nicht mehr erreichbar.

Wussten Sie schon …?

Wenn Sie den eBlocker im Menü Einstellungen > Netzwerk auf individuellen Netzwerkmodus umstellen, übernimmt er den DHCP-Server in Ihrem Netzwerk und behandelt alle gefundenen IP-Adressen als feste Adressen.

Sie können dann selbst bestimmen, ob ein Gerät die feste IP-Adresse behalten soll, oder der DHCP-Server des eBlockers diesem Gerät eine IP-Adresse aus dem DHCP-Bereich zuweisen soll.
