Wie binde ich einen eigenen DNS Server ein?

Mit der DNS Funktion vom eBlocker haben Sie nicht nur die Möglichkeit externe DNS Server einzubinden, Sie können auch einen eigenen Do It Yourself (DIY) DNS Server einbinden.

Wenn Sie bereits einen DIY DNS Server betreiben, sollten Sie als erstes sichergehen, dass dieser in Ihrem Router als DNS Server hinterlegt ist. Aktivieren Sie nun die DNS-Funktion in Ihrem eBlocker unter Einstellungen > Menü "DNS-Firewall". Wählen Sie die Option "Internet Provider" aus.
