Das eBlocker Zertifikat löschen

Das eBlocker (HTTPS) Zertifikat unter Windows löschen

Öffnen Sie das Dialogfeld "Ausführen" mittels der Tasten [Windows] und [R]. Geben Sie in die Befehlszeile das Kommando certmgr.msc ein und wählen Sie "OK" beziehungsweise drücken Sie die [Eingabetaste]. Es öffnet sich der Speicher für die Zertifikate. Wählen Sie in der Baumansicht links den zweiten Eintrag "Vertrauenswürdige Stammzertifizierungsstellen" und zeigen Sie die Inhalte im Unterordner "Zertifikate" an. Suchen Sie nun auf der rechten Seite das passende Zertifikat aus, das mit eBlocker... bezeichnet ist und klicken Sie das Zertifikat mit der rechten Maustaste an.Wählen Sie die Option "Löschen". Lassen Sie sich durch die abschreckende Warnung nicht aufhalten, das eBlocker Zertifikat können Sie jederzeit neu installieren, wenn es wieder benötigt wird. Das Fenster des Zertifikatsmanagers können Sie dann schließen.

Das eBlocker (HTTPS) Zertifikat aus Mozilla Firefox löschen

Öffnen Sie die Einstellungen des Firefox Browsers und wählen Sie links "Datenschutz & Sicherheit". Auf der rechten Seite unten finden Sie unter der Überschrift Zertifikate die Schaltfläche "Zertifikate anzeigen". Klicken Sie auf diese Schaltfläche. Im nun geöffneten Fenster Zertifikatverwaltung wählen Sie den Tab/Reiter Zertifizierungsstellen. Markieren Sie in der Liste den Eintrag für das eBlocker Zertifikat und klicken Sie auf die Schaltfläche "Löschen oder Vertrauen entziehen...".Bestätigen Sie trotz der Warnung den Abschluss des Vorganges. Das eBlocker Zertifikat können Sie jederzeit neu installieren, wenn es wieder benötigt wird. Ein Klick auf "OK" schließt die Zertifikatsverwaltung.

Das eBlocker (HTTPS) Zertifikat von einem Apple macOS Gerät löschen

Öffnen Sie die Schlüsselbundverwaltung unter Programme/Dienstprogramme. Wählen Sie links Schlüsselbund "System" und die Kategorie "Zertifikate" aus. Wählen Sie in der rechts erscheinenden Liste das eBlocker Zertifikat mit der rechten Maustaste aus und klicken Sie im Kontextmenü auf "löschen". Eventuell werden Sie nach dem Administrator-Passwort gefragt.
Schließen Sie dann das Fenster der Schlüsselbundverwaltung.

Das eBlocker (HTTPS) Zertifikat von einem Apple iOS Gerät löschen

Öffnen Sie die iOS Einstellungen und klicken Sie links auf "Allgemein" und anschließend rechts auf "Info". Ganz unten auf der Seite wählen Sie "Zertifikatsvertrauenseinstellungen".
Dort finden Sie das hinterlegte eBlocker-Zertifikat und können es jetzt deaktivieren.

Das eBlocker (HTTPS) Zertifikat von einem Android Gerät löschen

Beispiel ZTE + Android 5.0.2

Rufen Sie das Menü für die Telefon-Einstellungen auf und wählen Sie dort den Menüpunkt "Sicherheit". Klicken Sie auf den Menüpunkt "Vertrauenswürdige Anmeldedaten" und anschließend auf den Reiter/Tab Nutzer. Tippen Sie einmal auf den Eintrag der mit "eBlocker" beginnt und scrollen Sie im Zertifikat ganz nach unten. Dort tippen Sie auf "ENTFERNEN". Bestätigen Sie die anschließende Abfrage.

Beispiel Samsung Galaxy S8 plus [T-Mobile] + Android 8.0.0

Rufen Sie das Menü für die Telefon-Einstellungen auf und wählen Sie dort den Menüpunkt "Gerätesicherheit" aus. Wählen Sie "Andere Sicherheitseinstellungen" und dort "Sicherheitszertifikate anzeigen". Auf dem Reiter / Tab "BENUTZER" tippen Sie auf das Zertifikat das mit eBlocker in Verbindung steht und wählen Sie unten auf der Seite "entfernen".

Beispiel Sony Smartphone - Android 8.x

Öffnen Sie das Menü für die Telefon-Einstellungen und wählen Sie anschließend "Bildschirmsperre und Sicherheit" sowie "Verschlüsselung und Anmeldedaten". Sowohl unter dem Menüpunkt "Vertrauenswürdige Anmeldedaten" auf dem Reiter "Nutzer (Web)", als auch unter "Nutzeranmeldungen (VPN & Apps)" können eBlocker Zertifikate zum Entfernen liegen.
