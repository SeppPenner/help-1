Fehlermeldung beim Anmelden auf einer Webseite.

Sie wollen sich bei einer Webseite anmelden, bekommen aber eine Fehlermeldung (ERR_ZERO_SIZE_OBJECT). Der Grund dafür kann der „HTTP Referrer Header“ sein, den Sie in dem eBlocker Menü unter „Blocker“ > Reiter "Erweiterte Einstellungen" entfernt haben. Viele Webseiten brauchen die HTTP Referrer Header Information, um Sie beim Anmelden zu identifizieren. Deaktivieren Sie die „HTTP Referrer Header entfernen“ Funktion und melden Sie sich noch mal auf der Webseite an.
