Sehr langsame Netzwerkverbindung

Wenn Sie eine sehr langsame Netzwerkverbindung bemerken, überprüfen Sie, ob Ihr Gerät über das Tor-Netzwerk ins Internet geht.

eBlocker Dashboard

Das eBlocker Dashboard können Sie wie folgt aufrufen:

  • Öffnen Sie im Browser die Seite setup.eblocker.com
  • Oder öffnen Sie im Browser die Seite eblocker.box
  • Oder öffnen Sie im Browser die IP-Adresse Ihres eBlockers und geben dahinter „/dashboard“ ein (Beispiel: 192.168.10.210/dashboard)

Auf dem eBlocker Dashboard finden Sie eine Karte mit dem Namen "Anonymisierung". Sollte eine Verbindung zum Tor-Netzwerk bestehen, können Sie dort die Verbindung trennen. Überprüfen Sie, ob Sie nun wieder gewohnt, schnell im Internet unterwegs sind.

eBlocker Controlbar

Die eBlocker Controlbar können Sie wie folgt aufrufen:

Öffnen Sie die eBlocker Controlbar auf der Seite http://setup.eblocker.com, mit einem Klick auf das eBlocker Symbol. Sollte eine Verbindung zu Tor-Netzwerk, oder zu einem VPN-Server, aufgebaut sein, sollte das Schild über dem Punkt "IP-Anon" mit einem Karomuster ausgefüllt sein. Deaktivieren Sie die Tor- Netzwerk oder VPN-Server Verbindung und überprüfen Sie, ob Sie nun wieder gewohnt, schnell im Internet unterwegs sind.

Das Tor-Netzwerk ist ein nicht-kommerzieller Zusammenschluss von Menschen auf der ganzen Welt, die kostenlos Netzwerk-Infrastruktur bereitstellen, damit Nutzer Ihre IP-Adresse beim Surfen im Internet anonymisieren können. Das Tor-Netzwerk garantiert jedoch keine Verfügbarkeit oder Mindestbandbreite.

Wenn Sie dennoch eine sehr langsame Netzwerkverbindung bemerken, überprüfen Sie mit einem anderen Gerät, ob dieses Problem bei Ihnen im gesamten Netzwerk besteht. Wenn dem so ist, kann es sein, dass Ihr Router nicht mit dem automatischen Netzwerkeinstellungen des eBlockers zusammenarbeitet. In diesem Fall können Sie den eBlocker in seine individuellen Einstellungen umstellen, so dass der eBlocker den DHCP-Server in Ihrem Netzwerk übernimmt. Bei Ihrem Router können Sie den DHCP-Server dann deaktivieren.

Wie man den eBlocker in wenigen Schritten umstellt, können Sie hier in dem Artikel „Individuelle Netzwerkeinstellung“ nachlesen.
