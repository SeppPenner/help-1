Probleme mit Speedport W723V / W724V / W921V

Die Speedport Modelle W723V, W724V und W921V sind nur mit dem eBlocker kompatibel, wenn Sie an Ihrem Speedport den DHCP Server deaktivieren und den eBlocker individuell einstellen, sowie den DHCP Server des eBlockers nutzen. Eine Anleitung dazu finden Sie in unserer Knowledge Base.

Unsere Nutzer im Forum berichten über eine bessere Stabilität des eBlockers, wenn Sie in den Netzwerkeinstellungen unter Windows den IPv6 Support deaktivierten.
