IPv6 für das interne Netzwerk deaktivieren.

Wir empfehlen, das IPv6 Protokoll für das interne Netzwerk (Heimnetz) zu deaktivieren, da der eBlocker nicht IPv6-fähig ist. Bei einigen Routern ist dies jedoch nicht möglich. Sollte ihrer dazu gehören, können Sie das Protokoll an einzelnen Geräten deaktivieren (ist bei mobilen Geräten wie Smartphones und Tablets jedoch selten möglich).

IPv6-Deaktivierung unter macOS:

  • Systemeinstellungen > Netzwerk > aktive Verbindung > orangefarbenes Feld: weitere Optionen > Reiter: TCP/IP
  • "IPv6 konfigurieren" > "Nur Link-Local" auswählen

Oder de- bzw. aktivieren Sie IPv6 im Terminal ganz einfach mit einem Befehl:

  • Für eine normale Ethernet (zum Beispiel beim iMac): networksetup -setv6off Ethernet
  • Für das WLAN: networksetup -setv6off Wi-Fi
  • Für beispielsweise einen USB Ethernet Adapter: networksetup -setv6off "USB Ethernet"

Um IPv6 erneut zu aktivieren, geben Sie einen der folgenden Befehle ein:

  • networksetup -setv6automatic Ethernet
  • networksetup -setv6automatic Wi-Fi
  • networksetup -setv6automatic "USB Ethernet"

IPv6-Deaktivierung unter Windows:

Drücken Sie die Tastenkombination Windows + R, um den Ausführen-Dialog zu öffnen.

  • Geben Sie ncpa.cpl ein und bestätigen mit der Eingabetaste. Die Netzwerkverbindungen mit den einzelnen Netzwerkadaptern öffnen sich
  • Klicken Sie Ihre Netzwerkkarte mit der rechten Maustaste an und wählen Sie den Eintrag „Eigenschaften“ aus
  • Scrollen Sie etwas herunter und entfernen Sie den Haken vor dem Eintrag „Internetprotokoll, Version 6 (TCP/IPv6)“
  • Bestätigen Sie den Vorgang mit OK

Alternativ finden Sie Informationen zur Deaktivierung von IPv6 für die jeweiligen Windows-Versionen auf der folgenden Microsoft-Support-Seite:

https://support.microsoft.com/de-de/help/929852/guidance-for-configuring-ipv6-in-windows-for-advanced-users

IPv6-Deaktivierung unter Linux:

Bei den meisten Linux Betriebssystemen können Sie IPv6 ganz einfach in den Netzwerkeinstellungen deaktivieren. Zum Beispiel:

Einstellungen > Netzwerk > Aktuelle Verbindung bearbeiten > Reiter "IPv6" > Deaktivieren

Suchen Sie gegebenenfalls im Internet nach einer Anleitung für Ihr Linux-Betriebssystem.

IPv6-Deaktivierung im Firefox Browser (Desktop Version):

Der Firefox Browser ist so konfiguriert, dass er zunächst eine IPv6-Adresse (den sogenannten AAAA-Record) vom DNS-Server erfragt. Bleibt diese Anfrage erfolglos, wird nach einer IPv4-Adresse (dem sogenannten A-Record) gesucht.

Man kann dies ganz einfach deaktivieren.

  • In der Firefox-Adressleiste about:config eingeben
  • Über den Filter die Variable network.dns.disableIPv6 suchen
  • Mit einem Doppelklick den Wert von false auf true setzen

IPv6-Deaktivierung unter iOS:

  • Einstellungen
  • WLAN
  • Rechts beim WLAN auf das "i" klicken
  • Herunter scrollen bis zu DNS
  • Das Feld unter DNS anklicken
  • Einstellungen auf "Manuell" umstellen
  • Die IPv6 Adresse entfernen (Muster einer IPv6 Adresse: f80::e228:6aef:fe8b:cc59)
