Wie richte ich eine Portweiterleitung am Router ein?

Um eine Portweiterleitung einrichten zu können, müssen Sie in der Benutzeroberfläche Ihres Routers angemeldet sein. Einige Router benötigen für die Eingabe einer Portweiterleitung zudem die „erweiterte Ansicht“.

Im Nachfolgenden haben wir das Anlegen einer Portweiterleitung für die gängigsten Router beschrieben. Bitte beachten Sie, dass die Beschreibung gegebenenfalls leicht abweichen kann. Lesen Sie in dem Fall die Anleitung Ihres Routers oder schauen Sie in den FAQs des Router-Herstellers nach.

Fritzbox:

  • „Erweiterte Ansicht“ auswählen
  • Menü Internet > Freigaben > Reiter „Portfreigaben“
  • Button „Gerät für Freigaben hinzufügen“ anklicken
  • eBlocker auswählen
  • Button „Neue Freigabe“ anklicken
  • Portfreigabe anklicken
  • Im Reiter „Anwendung“ „Andere Anwendung“ auswählen
  • Bezeichnung frei wählen (zum Beispiel: eBlocker Mobile)
  • Protokoll = UDP
  • Port an Gerät = 1194
  • Port extern gewünscht = von 1194 bis 1194
  • „Freigabe aktivieren“ anklicken
  • und auf „OK“ klicken

TP-Link:

  • Menü Forwarding > Virtual Servers
  • Button „Add New... “
  • Service Port = 1194
  • Internal Port = 1194
  • IP Address = die IP-Adresse Ihres eBlocker
  • Protocol = All
  • Status = Enabled
  • Button „Save“ klicken

 TP-Link (alternativ):

  • Menü Erweitert > NAT > Port-Triggering
  • Button „(+) Hinzufügen“
  • Applikation =  frei vergeben (zum Beispiel: eBlocker Mobile)
  • Triggerport = 1194
  • Triggerprotokoll = UDP
  • Externer Port = 1194
  • Externes Protokoll = UDP
  • Aktivieren = (Häkchen setzen)
  • Button „OK“ klicken

 Netgear:

  • Reiter „Advanced“ auswählen
  • Menü Advanced Setup Einrichtung > Port Forwarding/Port-Triggering > Button „Add Custom Service“
  • Service Name frei vergeben (zum Beispiel: eBlocker Mobile)
  • Protokoll = UDP
  • External Starting Port = 1194
  • External Ending Port = 1194
  • Haken bei „use the same portrange for the internet port“ setzen
  • eBlocker IP-Adresse eingeben oder den eBlocker aus der Liste der Geräte wählen
  • Button „Apply“ anklicken

D-Link:

  • Reiter „Advanced“ auswählen
  • Im Menü „Virtual Servers“ auswählen
  • Den ersten unbenutzte Virtual Server aktivieren
  • Nun einen Namen vergeben (zum Beispiel: eBlocker Mobile)
  • Public= 1194 bis 1194
  • Protocol = UDP
  • Die IP-Adresse Ihres eBlocker bei „IP-Adresse“ eingeben
  • Private = 1194 bis 1194
  • Button „Einstellungen speichern“ anklicken

Apple Airport:

  • Starten Sie das macOS Airport Dienstprogramm
  • Wählen Sie den Reiter "Netzwerk" aus
  • Klicken Sie bei "Anschlusseinstellungen" auf den "+" Button
  • Vergeben Sie einen Namen in dem Feld Beschreibung (zum Beispiel "eBlocker Mobile")
  • Geben Sie für die öffentlichen TCP und UCP Ports 1194 ein
  • Geben Sie die eBlocker IP-Adresse in das Feld "Private IP Adresse" ein
  • Geben Sie für die privaten TCP und UCP Ports 1194 ein
  • Klicken Sie auf den Button "Speichern"
  • Klicken Sie auf den Button "Update"

Asus Router:

  • Starten Sie die Asus Merlin Software
  • Wählen Sie "Erweiterte Einstellungen" > "WAN" > Registerkarte "Port Trigger"
  • Klicken Sie auf den Auswahlknopf "ja"
  • " Beschreibung": Geben Sie einen Kurznamen ein, z.B. eBlocker.
  • Geben Sie 1194 in das Feld "Trigger Port" und 1194 in das Feld "Incoming Port" ein.
  • Wählen Sie UDP in den beiden Dropdown-Menüs "Protokoll".
  • Klicken Sie auf "Übernehmen".

Speedport W724V / W921V

  • Reiter „Internet“ > Menü „Portfreischaltung“
  • Button „Weitere UDP Umleitung anlegen“ anklicken
  • in den drei Felder je 1194 eingeben
  • den eBlocker oder dessen IP-Adresse im Dropdownfeld auswählen
  • und auf „OK“ / "Speichern" klicken
