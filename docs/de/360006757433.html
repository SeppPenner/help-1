Wie kann ich den DNS Cache leeren?

Wenn Sie eine Webseite aufrufen (Beispiel: www.eblocker.com), dann wird die Webseite von einem Domain Name Server im Internet übersetzt in eine IP-Adresse (138.68.124.96 ). Diese Information wird dann meist von Ihrem Gerät in einem DNS Cache zwischen gespeichert, damit ein erneuter Aufruf von www.eblocker.com schneller bearbeitet werden kann.

Wenn sich nun die IP-Adresse einer Webseite ändert, so kann es zu Problemen kommen. Der DNS Cache hat diese neue Information noch nicht und wird Sie zur alten IP-Adresse weiterleiten. In solchen Situationen hilft es meist einmal den DNS Cache zu leeren.

eBlocker

Öffnen Sie die eBlocker Einstellungen und gehen Sie in das Menü "DNS-Firewall". Klicken Sie unten auf der Seite auf den Button "Cache leeren", um den eBlocker DNS Cache zu leeren.

Windows

Öffnen Sie die Eingabeaufforderung (Command Promt). Sie finden die Eingabeaufforderung auch wenn Sie nach "cmd" suchen.
Geben in der Eingabeaufforderung den Befehl "ipconfig /flushdns" ein und der DNS-Cache von Windows wird gelöscht.

macOS 10.12.0 und höher

Öffnen Sie das Terminal. Sie finden das Terminal auch, wenn Sie in der Suche "terminal" eingeben.
Geben in im Terminal den Befehl "sudo killall -HUP mDNSResponder" ein und der DNS-Cache von macOS wird gelöscht.

iOS

Auf einem iPad, oder iPhone, reicht es meist einmal den Flugmodus zu aktivieren, um den DNS Cache zu löschen. Ein anderer Weg wäre das Gerät einmal neu zu starten.

Android

Auf einem Android Gerät kann man den DNS Cache löschen, in dem man das Gerät neu startet.

Linux (Ubuntu)

Um DNS-Caching-Services in Linux-Distributionen wie Ubuntu zu nutzen und den Cache zu leeren, benötigen Sie Administrator-Rechte.
Öffnen Sie ein Terminal und geben den Befehl "sudo /etc/init.d/dns-clean restart" ein und der DNS-Cache von Ubuntu wird gelöscht.

Browser Chrome / Chromium

Öffnen Sie den Browser und geben Sie in der Adresszeile "chrome://net-internals/#dns" ein. Klicken Sie dann aufn den Button "Clear host cache" um den DNS Cache vom Browser zu löschen.

Browser Firefox

Öffnen Sie den Browser und geben Sie in der Adresszeile "about:config" ein und suchen Sie nach "network.dnsCacheExpiration". Klicken Sie den Wert (60) doppelt an und setzen Ihn auf 0. Mit dem Klick auf "OK" wird dann der Browser Cache geleert. Danach können Sie den Wert wieder auf "60" setzten.
