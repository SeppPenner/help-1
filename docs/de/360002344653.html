eBlocker Hilfe > FAQs > HTTPS Funktion (SSL/HTTPS)

Mein Virenscanner bemängelt HTTPS / SSL Verbindungen.

Prüfen Sie, ob sie das eBlocker Zertifikat auch in Ihrem Betriebssystem hinterlegt haben.

Bei einigen Virenscannern muss man die Funktion "SSL Scanning", oder "HTTPS Scanning" deaktivieren.

Hier einige Beispiele:

Avast

  • Öffnen Sie die Einstellungen
  • Wählen Sie "Komponenten" aus
  • Klicken Sie auf "Anpassen" neben dem "Web-Schutz"
  • Entfernen Sie das Häkchen bei "HTTPS-Scanning aktivieren"


Avira (E-Mail-Schutz)

  • Öffnen Sie die Einstellungen
  • Klicken Sie auf Star > Einstellungen > System > Apps und anschließend auf Funktionen
  • Klicken Sie mit rechter Maustaste auf "Avira Antivirus" > Ändern > und auf Weiter
  • Entfernen Sie das Häkchen vom "E-Mail-Schutz" und klicken Sie Weiter, bis zum Ende

Bitdefender

  • Öffnen Sie die Einstellungen
  • Klicken Sie nun auf "Module"
  • Klicken Sie auf "Internet Schutz"
  • Entfernen Sie das Häkchen bei "SSL scannen"

oder

  • Öffnen Sie die Einstellungen
  • Wählen Sie im Menü "Schutz" aus
  • Klicken Sie auf die Einstellungen bei "Online-Gefahrenabwehren"
  • Deaktivieren Sie "Verschlüsselter Web-Scan"

ESET

  • Öffnen Sie die Einstellungen
  • Drücken Sie die F5 Taste für die "Erweiterten Einstellungen"
  • Klicken Sie nun auf "Web und E-Mail"
  • Klicken Sie auf "SSL/TLS" um die Einstellungen auf zuklappen
  • Klicken Sie nun auf den Schieberegler, um die Funktion zu deaktivieren
  • Klicken Sie auf OK

Ältere ESET Produkte

  • Öffnen Sie die Einstellungen
  • Drücken Sie die F5 Taste für die "Erweiterten Einstellungen"
  • Klicken Sie nun auf "Web und E-Mail"
  • Klicken Sie auf "Protokolfilterung"
  • Klicken Sie nun auf "SSL"
  • Wählen Sie "SSL-Protokoll nicht scannen" aus
  • Klicken Sie auf OK

GData (E-Mail-Prüfung)

  • Öffnen Sie die Einstellungen
  • Klicken Sie auf "AntiVirus"
  • Klicken Sie nun auf "E-Mail-Prüfung"
  • Entfernen Sie das Häkchen bei "SSL-Verbindungen prüfen"
  • Klicken Sie nun auf OK

Kaspersky

  • Öffnen Sie die Einstellungen
  • Klicken Sie auf "Erweitert"
  • Klicken Sie nun rechts auf "Netzwerk"
  • Wählen Sie bei dem Punkt "Untersuchung von sicheren Verbindungen" die Option "Sichere Verbindungen nicht untersuchen" aus
