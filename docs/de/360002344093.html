eBlocker DHCP Server und feste IP Adressen

Wenn Sie den eBlocker DHCP Server aktivieren, übernimmt der eBlocker alle gefundenen IP-Adressen aus Ihrem Netzwerk. Alle IP-Adressen sind als feste IP-Adressen im DHCP Server des eBlocker markiert. Über das Menü "Geräte" können Sie anschließend bestimmen, ob ein Gerät seine festen IP-Adressen behalten soll.
