Tipps zu dem eBlocker Dashboard

Das eBlocker Dashboard ist die Steuerzentrale für Ihr Gerät und Ihren eBlocker. Hier können Sie alle wichtigen Informationen einsehen und  Änderungen vornehmem.

Nach und nach werden wir immer mehr sogenannter Karten in dem Dashboard zur Verfügung stellen und somit den Funktionsumfang stark erweitern. Sie werden dann auch komplexere Konfigurationen des eBlocker für Ihr Gerät über das Dashboard erledigen können.

Das Dashboard rufen Sie mit der URL eblocker.box oder über die Seite setup.eblocker.com auf. Sie können auch die IP-Adresse Ihres eBlockers aufrufen und dahinter /dashboard schreiben ( Beispiel: 192.168.10.210/dashboard ).

Tipp: Erstellen Sie in Ihrem Brwoser ein Lesezeichen (Bookmark) für Ihr eBlocker Dashboard oder besser noch, legen Sie das Dashboard gleich als Startseite für Ihren Browser fest – so haben Sie alle Informationen immer schnell zu Hand.

Firefox – eBlocker Dashboard als Lesezeichen hinzufügen:
Öffnen Sie das eBlocker Dashboard und klicken Sie rechts in der Adresszeile des Browsers auf die drei Punkte. Wählen Sie „Lesezeichen hinzufügen" aus.

Firefox – eBlocker Dashboard als Startseite festlegen:
Öffnen Sie die Firefox Einstellungen und wählen Sie dann „Startseite" aus. Wählen Sie aus dem Menü „Homepage und neue Fester" „Benutzerdefinierte Adresse" aus und geben Sie die Adresse Ihres eBlocker Dashboards ein.

Chrome – eBlocker Dashboard als Lesezeichen hinzufügen:
Öffnen Sie das eBlocker Dashboard und klicken Sie rechts in der Adresszeile des Browsers auf den Stern. Legen Sie dort das eBlocker Dashboard als Lesezeichen an.

Chrome – eBlocker Dashboard als Startseite festlegen:
Öffnen Sie die Chrome Einstellungen und dann die Einstellungen der Einstellungen. Wählen Sie aus dem Menü „Beim Start" aus. Nun wählen Sie „Bestimmte Seite oder Seiten" aus und geben die Adresse Ihres eBlocker Dashboarda ein.

Opera – eBlocker Dashboard als Lesezeichen hinzufügen:
Öffnen Sie das eBlocker Dashboard und klicken Sie rechts in der Adresszeile des Browsers auf das Herz. Legen Sie dort Ihre eBlocker Übersicht als Lesezeichen an.

Opera – eBlocker Dashboard als Startseite festlegen:
Öffnen Sie die Opera Einstellungen und klicken Sie dann im Menü auf „Browser“. Wählen Sie beim Punkt „Beim Starten" nun die Option „Bestimmt Seite oder bestimmte Seiten öffnen" aus und geben die Adresse Ihres eBlocker Dashboards ein.

Windows Edge – eBlocker Dashboard als Lesezeichen hinzufügen:
Öffnen Sie das eBlocker Dashboard und klicken Sie rechts in der Adresszeile des Browsers auf den Stern. Legen Sie dort Ihre eBlocker Übersicht als Lesezeichen an.

Windows Edge – eBlocker Dashboard als Startseite festlegen:
Öffnen Sie die Edge Einstellungen, klicken Sie dann auf „Erweiterten Einstellungen anzeigen" und wählen Sie aus dem Dropdown-Menü „Startseite" den Punkt „Bestimmte Seite" aus. Dort geben Sie die Adresse Ihres eBlocker Dashboards ein.

Windows Internet Explorer – eBlocker Dashboard als Lesezeichen hinzufügen:
Öffnen Sie das eBlocker Dashboard und klicken Sie rechts in der Adresszeile des Browsers auf den Stern. Klicken Sie nun auf „Zu Favoriten hinzufügen" und legen Sie die eBlocker Übersicht als Lesezeichen an.

Windows Internet Explorer – eBlocker Dashboard als Startseite festlegen:
Öffnen Sie die Internet Explorer Einstellungen, klicken Sie dann auf „Internet Optionen" und wählen Sie den Reiter „Allgemein" aus. Geben die Adresse Ihres eBlolcker Dashboards im Feld Startseite ein.
