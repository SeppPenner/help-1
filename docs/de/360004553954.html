DIY eBlocker – Softwareinstallation

Laden Sie eine unserer letzten eBlockerOS Versionen herunter und schreiben Sie sie auf eine SD-Karte. Sie benötigen dafür eine 8Gb Class 10 SD-Karte, das eBlockerOS Image und ein Programm zum Schreiben des Images.

Laden Sie zunächst das eBlockerOS Image für Ihre Hardware herunter:

  • Raspberry Pi 2 und 3
  • Raspberry Pi3 B+
  • Banana Pi M2
  • Banana Pi M2+

Bitte laden Sie die Software nur, wenn sie mit unseren Lizenzbestimmungen (EULA) einverstanden sind.

Um das eBlockerOS Image auf eine SD-Karte schreiben zu können, benötigen Sie eine Software. Wir empfehlen das kostenlose und einfach zu bedienende Programm Etcher (https://etcher.io). Das Programm Etcher gibt es für Windows, macOS und Linux.

Legen Sie nun Ihre SD-Karte in Ihren Rechner ein und starten Sie das Programm Etcher. Wählen Sie das heruntergeladene Image aus und schreiben Sie das Image auf die SD-Karte.

Entnehmen Sie die SD-Karte und legen Sie diese in Ihren Raspberry, oder Banana Pi ein. Verbinden Sie den Raspberry, oder Banana Pi, mit Ihrem Netzwerk und anschließend mit dem Netzteil. Anschließend wird der eBlocker starten, was ca. 5 Minuten dauern kann.

Öffnen Sie einen Browser und rufen Sie die Seite http://setup.eblocker.com auf. Oben rechts im Browser sehen Sie nun das eBlocker Symbol. Klicken Sie auf das Symbol und anschließen in der eBlocker Controlbar auf „Registieren“.

Damit Sie gleich mit Ihrem eBlocker loslegen können, bieten wir Ihnen eine kostenlose 30 Tage Testlizenz an. Diese können Sie auf der Webseite bestellen. Kurz darauf bekommen Sie eine Mail von uns, in der Sie Ihre persönliche Testlizenz finden. Mit dieser Testlizenz können Sie den eBlocker nun registrieren und 30 Tage lang testen.

Wichtiger Tipp: Nachdem Sie den eBlocker registriert haben, machen Sie ein Update auf das aktuelle eBlockerOS. Das Update kann bis zu 10 Minuten dauern.
