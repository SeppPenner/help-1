Kein eBlocker Dashboard

Beim Aufruf von setup.eblocker.com, oder eblocker.box sollten Sie im Browser das eBlocker Dashboard sehen.

Wenn Sie statt dessen eine Webseite ohne einem eBlocker Icon sehen, dann überprüfen Sie zunächst folgendes:

  • Ist das Netzteil richtig zusammen gebaut?
  • Ist das Netzwerkkabel richtig mit dem eBlocker LAN Port verbunden?
  • Ist das Netzwerkkabel richtig mit dem Router verbunden?
  • Wenn alles richtig angeschlossen ist, sollten beim eBlocker nach einigen Sekunden die beiden LEDs leuchten / blinken.

Wenn beim eBlocker die beiden LEDs am LAN Port blinken, öffnen Sie die Einstellungen Ihres Routers. Unter den Netzwerkeinstellungen sollten Sie den eBlocker als neues Gerät sehen. Ihr Router hat den eBlocker eine IP-Adresse gegeben. Notieren Sie sich diese IP-Adresse.

Öffnen Sie einen neuen Browser Tab. Geben Sie die notierte IP-Adresse ein und fügen Sie am Ende der IP-Adresse :3000 hinzu (Beispiel: 192.168.0.26:3000). Jetzt sollten Sie die eBlocker Einstellungen sehen.

Wenn Sie die eBlocker Einstellungen nicht sehen, probieren Sie die eBlocker Notfall IP-Adresse. 169.254.94.109:3000

In den eBlocker Einstellungen finden Sie den Menüpunkt "Geräte" (Englisch: Devices). Hier finden Sie alle vom eBlocker gefundene Geräte in Ihrem Netzwerk. Ihr Gerät, mit dem Sie in den eBlocker Einstellungen sind, wird in dieser Liste an erster Stelle angezeigt . Klicken Sie auf die IP-Adresse Ihres Gerätes und überprüfen Sie in den Einstellungen, ob der eBlocker für dieses Gerät aktiviert ist.

Tipp: Legen Sie sich ein Lesezeichen für Ihr eBlocker Dashboard an, so haben Sie immer schnellen Zugriff auf das Dashboard.
