Hilfe, eine Webseite wird nicht mehr angezeigt.

Je mehr Blockerlisten man einsetzt, um so eher kann es vorkommen, dass eine Webseite vom Browser nicht geladen wird.

Angenommen Sie rufen zum Beispiel im Browser die Webseite facebook.com auf und sehen im Anschluss im Browser die Meldung „Seite kann nicht geladen werden“, dann pausieren Sie den eBlocker in dem eBlocker Dashboard und versuchen diese Seite erneut zu laden. Wenn dies funktioniert, dann liegt das eventuell an einer der importieren Blockerlisten.

Beenden Sie nun wieder die Pause des eBlockers im eBlocker Dashboard. Auf dem eBlocker Dashboard finden Sie die Karte „Blocker“. Hier können Sie nun die Webseite (Domain) erlauben.

Beim nächsten Aufruf der Webseite sollten Sie nun diese auch wieder sehen können.
