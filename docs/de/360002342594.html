Es geht auch ohne einen DHCP Server

Sie können auf einen DHCP Server in Ihrem Netzwerk verzichten, wenn Sie bei jedem Gerät in Ihrem Netzwerk die Netzwerkeinstellungen manuell einrichten. Der Nachteil dieser Methode ist jedoch, dass jedes neue Gerät im Netzwerk anschließend manuell konfiguriert werden muss.

Worauf Sie achten müssen:

  • Benutzen Sie die Netzwerkmaske 255.255.255.0
  • Alle Geräte müssen in einem IP-Bereich sein (als Beispiel: 192.168.1.1 bis 192.168.1.254)
  • Eine IP-Adresse darf nur einmal im Netzwerk vergeben werden
  • Alle Geräte die geschützt werden sollen, müssen die IP-Adresse des eBlockers als Gateway erhalten
  • Alle Geräte die nicht geschützt werden sollen, erhalten die IP-Adresse des Routers als Gateway
  • Der eBlocker bekommt die IP-Adresse des Routers als Gateway

Wir empfehlen alle vergebenen IP-Adressen zu notieren.
