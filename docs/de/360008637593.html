Was ist fragFINN?

Ab der eBlockerOS Version 2 haben wir einen zusätzlichen Schutz für Ihre Kinder durch eine umfangreiche Schutzliste von fragFINN eingebaut. fragFINN ist eine Internet Suchmaschine für Kinder, deren Ergebnisse speziell für kleinen Nutzer ausgewählt werden.

Wenn Sie ein neues Jugendschutzprofil anlegen, können Sie zum Beispiel die Zugriffsbeschränkungen wie folgt einstellen:

  • Wählen Sie den Grundsatz "Folgende Kategorien sind erlaubt, alle anderen Webseiten sind verboten" aus.
  • Wählen Sie fragFINN aus und speichern Sie die Einstellung.

Wenn Sie dieses Jugendschutzprofil nun einem Benutzer zuweisen und dem Benutzer im Anschluss ein bestimmtes Gerät zuordnen, kann dieser Benutzer auf diesem Gerät nur noch auf die von fragFINN ausgewählten Webseiten zugreifen.

Zusätzlich haben wir zu der Meldung des eBlockers ein fragFINN Suchfeld eingebaut.
Sollten Sie Ihre eBlocker Oberfläche auf Englisch eingestellt haben, so sehen Sie hier die Suche von DuckDuckGo.
