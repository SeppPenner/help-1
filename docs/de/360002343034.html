Fritzbox 7490 - Das WLAN ist sehr langsam.

Die Fritzbox 7490 hat einen bestätigten Bug, welcher dafür sorgt, dass Ihr WLAN nach dem Anschluss des eBlockers langsamer ist, als Sie es gewohnt sind. Dies ist kein Bug im eBlocker. Andere Fritzbox Modelle sind nicht von dem Bug betroffen.

Das Problem tritt auf, wenn Sie den Netzwerk Modus des eBlockers auf Automatisch belassen. Eine Anleitung zur Umstellung auf die Individuellen Einstellungen des eBlockers finden Sie hier.

Eine andere Lösung für Fritzbox 7490 Nutzer wäre einen WLAN Access Point in Ihrem Netzwerk zu intrigieren.

Mehr Informationen finden Sie in unserer Knowledge Base.
