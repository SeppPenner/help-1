Was kann gegen Abbrüche der VPN Verbindung getan werden?

Bei Verwendung von VPN-Verbindungen, trennen manche Anbieter inaktive Verbindungen nach einer zu kurzen Zeitspanne im Leerlauf.

Dies kann vom eBlocker unbemerkt bleiben, wenn die Verbindung nicht aktiv überwacht wird. Wenn Verbindungsprobleme auftreten, können Sie die aktive Überwachung aktivieren.

Wählen Sie im eBlocker-Menü einfach "IP-Anonymisierung" > "VPN-Netzwerk" und öffnen Sie die Details einer VPN-Verbindung, indem Sie auf die entsprechende Tabellenzeile klicken.

Unter "Aktive Überwachung der VPN-Verbindung" können Sie eine der folgenden Optionen wählen:

Deaktiviert (Standard) - es entsteht kein zusätzlicher Aufwand für die Aufrechterhaltung der VPN-Verbindung, wenn Ihr VPN-Provider eine bidirektionale Keep-Alive-Überwachung der VPN-Verbindung anbietet, eine zusätzliche aktive Überwachung ist nicht erforderlich.

Regelmäßiger Ping zum VPN-Anbieter - es wird eine minimale Menge an Traffic an Ihren VPN-Provider weitergeleitet, um die Verbindung aufrechtzuerhalten.

Regelmäßiger Ping zu eBlocker - Ihr eBlocker sendet eine minimale Menge an Traffic an unsere Domain www.eblocker.com, damit Ihr eBlocker die VPN-Verbindung aufrecht erhalten kann.

Regelmäßiger Ping zu anderem Host - Sie können eine Domain oder IP-Adresse eingeben, unter der der eBlocker minimalen Netzwerkverkehr senden soll, um die Verbindung aufrechtzuerhalten.
