eBlocker Hilfe > FAQs > Mobile Funktion

Wichtige Voraussetzungen

Sie sollten eine IPv4 Verbindung von Ihrem Internetprovider gestellt bekommen. eBlocker Mobile funktioniert nicht mit einer IPv6 Verbindung.

In dem Router Ihres Netzes, in dem der eBlocker angezeigt wird, sollte zudem eine Portweiterleitung von Port 1194 (Protokoll UDP) auf die IP Adresse des eBlockers eingerichtet werden.

Um einen festen DNS Namen für die eBlocker-Mobile Funktion zu haben, ist DynDNS optional sehr hilfreich.

Wenn Sie mehrere Router haben sollten und Ihr eBlocker am zweiten Router angeschossen sein sollte, dann müssen Sie vom ersten zum zweiten Router den Port 1194 weiterleiten und vom zweiten Router anschließend den Port 1194 an den eBlocker weiterleiten.

Sollten Sie einen Router besitzen welcher kein Portforwarding erlaubt, dann können Sie die Mobile Funktion des eBlocker nicht nutzen.

Die eBlocker DNS-Firewall Funktion sollte aktiviert sein.
