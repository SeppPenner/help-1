eBlocker Hilfe > FAQs > HTTPS Funktion (SSL/HTTPS)

Die Erstellung des Zertifikats dauert sehr lange.

Sie haben die HTTPS Funktion aktiviert und sehen den „Achtung“ Hinweis. Unten rechts finden Sie den Button „HTTPS Aktivieren“. Dieser ist grau, was bei älteren eBlockerOS Versionen ein bekannter UI Bug war. Sie können auf den Button klicken und müssen nicht warten, bis das Zertifikat erstellt wurde.
