Kein Internet, aber der eBlocker ist erreichbar

In seltenen Fällen kann es vorkommen, dass keine Verbindung zum Internet vorhanden ist. Der Grund dafür könnte ein Absturz einer essentiellen eBlocker Unterroutine sein, oder eine Einstellung bei der eBlocker DNS-Funktion.

Prüfen Sie in diesem Fall folgendes:

  • Ist Ihr Router im Browser mit seiner IP-Adresse zu erreichen?
  • Können Sie die eBlocker Einstellungen im Browser aufrufen wenn Sie die IP-Adresse vom eBlocker eingeben (bei älteren eBlockerOS Versionen geben Sie bitte die eBlocker IP-Adresse und :3000 ein (Beispiel: 192.168.1.240:3000)?

Wenn Sie beides mit "ja" beantworten können, gehen Sie bitte in die eBlocker Einstellungen > Menü "System" > Reiter "Neustart und Herunterfahren" und starten Sie den eBlocker neu (Button "Neustart").

Ihr eBlocker wird erst herunterfahren und dann neu starten. Der Vorgang dauert ca. fünf Minuten. Nach dem Neustart sollten Sie wieder wie gewohnt das Internet nutzen können.

Wenn der Vorgang nicht zum gewünschten Erfolg führt, gehen Sie bitte in die eBlocker Einstellungen > Menü "IP-Anonymisierung" > Reiter "DNS".

Überprüfen Sie bitte folgendes:

  • Ist die eBlocker DNS Funktion aktiviert?
  • Nutzen Sie die Einstellungen "Benutzerdefinierte Liste von externen DNS Servern" und haben dort nur einen DNS Server (zum Beispiel Ihren Router) angegeben?

Wenn Sie beides mit "ja" beantworten können, ändern Sie die eBlocker DNS bitte wie folgt:

Wählen Sie entweder die Option "Internet Provider", welche den DNS-Server in Ihrem Netzwerk verwendet.
Oder nutzen Sie weiterhin die Option "Externe DNS Servern" und geben Sie dort mehr als einen DNS-Server an.

Eine Liste von DNS-Server finden Sie zum Beispiel hier.

Sollte dies öfter vorkommen, kontaktieren Sie bitte das eBlocker Support-Team und senden Sie den eBlocker Diagnosebericht mit. Diesen finden Sie in den eBlocker Einstellungen > Menü "System" > Reiter "Diagnosebericht". Der Bericht wird in Ihrem Downloadordner oder in dem von Ihnen bestimmten Ordner gespeichert.
