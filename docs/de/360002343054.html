Probleme mit der Fritzbox 7490 / 7360 / 3490 / 3272

Diese Fritzbox Modelle sind nur mit dem eBlocker kompatibel, wenn Sie an Ihrer Fritzbox den DHCP Server deaktivieren und den eBlocker individuell einstellen, sowie den DHCP Server des eBlockers nutzen. Eine Anleitung dazu finden Sie in unserer Knowledge Base.
