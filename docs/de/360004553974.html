DIY eBlocker – Unterstützte Hardware

Technisch versierte Nutzer können sich ihren eBlocker selbst zusammenbauen (DIY – Do It Yourself). Einfach das eBlockerOS kostenlos runterladen, eine kostenlose 30 Tage Demolizenz anfordern und sofort anonym surfen. Sie benötigen nur einen Raspberry Pi oder Banana Pi sowie eine 8Gb Class 10 SD-Karte.

Das eBlockerOS unterstützt zur Zeit:

  • Raspberry Pi 2 Version 1.1 und Version 1.2
  • Raspberry Pi 3
  • Raspberry Pi 3 B+
  • Banana Pi M2
  • Banana Pi M2+

Wichtig: Stellen Sie sicher, dass Ihr Netzteil 10 Watt oder 2A liefert. Die meisten Standard USB Netzteile für Computer und Router liefern dies nicht. Dies kann unter Umständen zu Problemen und Fehlern führen.
