eBlocker Hilfe > FAQs > Anon Funktion (VPN)

Mit einer VPN App sieht man kein eBlocker Icon.

Die meisten VPN Anbieter bieten Ihnen auch eine App zum Start der VPN Verbindung an, oder verweisen auf eine App eines Drittanbieters. Wenn Sie nun statt des eBlockers eine App nutzen, um eine VPN Verbindung aufzubauen, werden Sie das eBlocker Icon nicht im Browser sehen können. Solche Apps leiten den Netzverkehr direkt an den VPN Server und können somit den eBlocker umgehen.
