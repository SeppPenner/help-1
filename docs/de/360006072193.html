Running with errors / Database stopped

Sie sehen beim Start Ihres eBlocker, oder beim Aufrufen der eBlocker Benutzeroberfläche, die Fehlermeldung "eblocker is running with errors" oder "database stopped"?

In diesem sehr seltenen Fall versuchen Sie bitte, ob das Wiederherstellen der eBlocker Werkseinstellungen dieses Problem beheben kann.
Eine Anleitung zum für das Wiederherstellen der Werkseinstellungen finden Sie hier.

Sollte Ihr eBlocker weiterhin diese Fehlermeldung anzeigen, so erstellen Sie wenn möglich mit der Hilfe eines USB 2.0 Stick einen kleinen Statusbericht. Diese Datei senden Sie dann bitte mit einer kurzen Beschreibung Ihres Problems an unser Support Team (support@eblocker.com)

Wie kann ich einen Status-Report auf einen USB-Stick (USB 2.0 / FAT16 oder FAT32 formatiert) schreiben lassen?

(Achtung: löschen Sie gegebenenfalls vorher vom Stick die Datei "eblocker-factory-reset.txt")

  • Trennen Sie den eBlocker vom Strom, lassen Sie aber das Netzwerkkabel am eBlocker und am Router angeschlossen.
  • Stecken Sie einen USB-Stick an den unteren USB Port des eBlockers.
  • Verbinden Sie den eBlocker mit der Stromversorgung.
  • Warten Sie mindestens 3 Minuten.

Danach können Sie den USB-Stick abziehen und am Computer anstecken. Auf dem Stick sollte sich nun eine Datei "eblocker-status.html" befinden. Bitte senden Sie diese Datei an unseren Support (support@eblocker.com).
