Where can I find block lists on the Internet?

There are many block lists available in the Internet, but most of them are hard to find. The following two websites offer a collection of block lists.

At https://firebog.net/ you will find mainly domain and ect.hosts lists. Always pay attention to the beginning of the lists. If you find an IP address at the beginning (127.0.0.1 or for example 0.0.0.0), it is an ect.hosts list.

On https://filterlists.com/ you can find a lot of filter lists. Here you can for example search for "AdBlock" lists, which you can import into the eBlocker Pattern Blocker. Make sure that you select the "easylist" format for such lists.
