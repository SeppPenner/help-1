Information on radios, loudspeakers, printers, NAS.

For some devices, we recommend to de-activate the eBlocker. These include Internet radios, network loudspeakers, network printers, network attached storage (NAS), network cameras and IP telephones.Â 

In the eBlocker Settings> Devices, you can deactivate the eBlocker for these devices.
